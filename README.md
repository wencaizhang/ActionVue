## Installation

Install the package in your project directory with:

```bash
# with npm
npm install actionvue --save

# or yarn
yarn add actionvue
```

## Usage

Here is a quick example to get you started.

```js
// src/main.js
import Vue from 'vue';
import actionvue from 'actionvue';
import App from './App'

Vue.use(actionvue);

new Vue({
  el: '#app',
  router,
  components: { App },
  render: h => h(App)
})
```
